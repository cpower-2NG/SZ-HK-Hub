## 二、半日游推荐路线

### 路线1：经典港岛游（适合下午+晚上）
尖沙咀天星码头 → 天星小轮 → 中环 → 太平山顶（日落+夜景）→ 兰桂坊/铜锣湾

### 路线2：九龙漫步游（适合半天）
西九龙文化区 → 星光大道 → 天星小轮 → 庙街夜市

### 路线3：自然探索游（适合全天上午）
东涌 → 昂坪360 → 大澳渔村 → 返回

## 三、跨境游客实用贴士

- **八达通**：必备，口岸港铁站有售（HK$50按金+首充HK$100起）
- **支付宝香港**：大部分商户和交通已支持
- **现金**：建议兑换HK$300-500以备小店使用
- **4G/5G上网**：淘宝可买香港流量卡（约¥15/天），或开通手机漫游
- **充电宝**：香港公共充电站少，建议自备
- **购物退税**：香港没有消费税/增值税，无需办理退税
- **货币兑换**：口岸汇率较差，市区找换店较优（如百年、许氏）

## 四、从比赛地点出发的旅游建议

- **港科大 → 西贡 + 将军澳**：比赛后可顺路去西贡吃海鲜，将军澳看日落
- **港大/中环 → 山顶 + 兰桂坊**：比赛后步行可达
- **九龙塘/旺角 → 庙街 + 维港夜景**：交通30分钟内可达
- **沙田/中文大学 → 沙田马场 + 大埔**：新界深度游
