# 香港科技大学（HKUST）出行指南

## 一、位置

香港科技大学位于**九龙清水湾**大埔仔，不在地铁沿线，需巴士/小巴接驳。

## 二、从深圳前往科大路线

### 方案A：罗湖/福田口岸 + 港铁 + 巴士（最经济）
1. 过关后乘港铁东铁线 → **九龙塘站**（约35分钟，约HK$40）
2. 转观塘线 → **彩虹站**（约10分钟）
3. 彩虹站C1出口 → 乘**11M/11号绿色小巴**或**91/91M巴士** → 科大（约20分钟， HK$7-8）
- **总耗时**：约1.5小时 ｜ **总费用**：约HK$50

### 方案B：深圳湾口岸 + 跨境巴士 + 转乘
1. 乘跨境巴士 → 旺角（约50分钟，HK$45-55）
2. 旺角 → 乘**小巴1A**或**巴士91** → 科大（约50分钟）
- **总耗时**：约2小时 ｜ **总费用**：约HK$55-70

### 方案C：西九龙高铁（最快）
1. 深圳北/福田 → 西九龙高铁站（14-24分钟，CNY 68-90）
2. 步行至柯士甸站 → 乘港铁至彩虹站 → 转小巴（约50分钟）
- **总耗时**：约1.5小时 ｜ **总费用**：约CNY 90 + HK$20

## 三、科大周边

| 地点 | 距离 | 交通 |
|------|------|------|
| 将军澳 | 15分钟 | 小巴11M |
| 坑口 | 10分钟 | 小巴11 |
| 彩虹/九龙湾 | 20分钟 | 91/91M巴士 |
| 西贡市中心 | 15分钟 | 小巴 |

## 四、旅游建议

科大位于清水湾半岛，推荐顺路游玩：
1. **西贡**：海鲜街、地质公园、出海游（距科大15分钟小巴）
2. **清水湾泳滩**：科大步行可达，香港水质最好的海滩之一
3. **狮子会自然教育中心**：免费，适合半天游
4. **将军澳海滨长廊**：骑行、日落

## 五、注意事项

- 科大地处山上，建议穿舒适的鞋
- 校园内有餐厅和便利店，可用八达通/支付宝
- 比赛完可从坑口站返回（比彩虹站更近）
- 夜间小巴班次减少，注意末班车时间（约23:00-00:00）
