# 香港海关旅客清关指南

## 一、红绿通道系统

香港各口岸入境大堂设有红绿通道：

| 通道 | 适用情况 |
|------|----------|
| 🟢 绿通道 | 没有需要申报的物品 |
| 🔴 红通道 | 携带需申报物品（超额免税品、禁运/受管制物品、≥HKD 120,000 现金等）|

> 走绿通道代表向海关声明没有携带任何需申报物品。若被查出携带需申报物品，可被检控或罚款。

## 二、免税优惠（自2010年8月1日起生效）

### 烈酒
- **年满18岁**旅客可免税携带：**1升**酒精浓度超过30%的烈酒
- 持有香港身份证者须离港**超过24小时**

### 烟草
- **年满18岁**旅客可免税携带以下其中一种：
  - **19支**香烟；或
  - **1支**雪茄，或**25克**雪茄；或
  - **25克**其他制成烟草

### 跨境司机
- 跨境司机**不享有**免税优惠

## 三、货币及不记名票据申报

根据《跨境运大量货币及不记名票据条例》（第629章）：

- 任何入境人士如管有总值**高于HKD 120,000**的货币或不记名票据（CBNI），**必须**经红通道向海关人员申报
- CBNI 包括：现金（港币及外币）、旅行支票、不记名支票、本票、不记名债券、邮政汇票
- 申报时**不需**将外币兑换为港币，直接填原币种金额即可
- 出发旅客如被海关人员查问，也必须披露是否管有大量CBNI

### 申报方式
- **电子申报**：可提前通过「旅客信息电子提交表格」填写，生成QR码
- **纸质申报**：各口岸红通道备有纸质申报表(CED412)

### 罚则
- **初犯**：可缴付HK$2,000解除法律责任（前提：未曾被裁定犯有洗钱或恐怖分子融资罪行，且CBNI非犯罪得益）
- **其他情况**：刑事检控，最高罚款**HK$500,000**及监禁**2年**

## 四、禁运/受管制物品

以下物品未持有有效牌照/许可证/卫生证明书情况下**禁止**携带进出香港：

| 类别 | 常见物品 |
|------|----------|
| 危险药物 | 大麻、可卡因、海洛因、冰毒等 |
| 精神药物 | 安眠药（部分需医生证明） |
| 武器弹药 | 枪支、弹药、电击器、催泪喷雾 |
| 烟花炮竹 | 各类烟花爆竹 |
| 战略物品 | 高科技设备、加密技术产品 |
| 未切割钻石 | 毛坯钻石（受金伯利进程管制） |
| 动植物 | 活体动物、濒危物种（象牙、犀牛角等） |
| 肉类/家禽/蛋 | 生肉、鸡蛋（内地常买到但香港禁止入境） |
| 配方奶粉 | 每人限带**2罐**（1.8公斤），16岁以上方可携带 |
| 大米 | 个人使用限15公斤 |
| 无线电设备 | 未获发牌的无线电发射设备 |

> 违例者将被检控，物品会被充公。

## 五、应课税品（烟酒以外）

香港是自由港，只对四类商品征收消费税：
- 烈酒（酒精浓度>30%）
- 烟草
- 烃油（汽油、柴油等）
- 甲醇

其他商品免关税，但不免管制（上述第四类）。
